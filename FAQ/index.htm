FAQ example

Demonstrating simple use of Javascript UI

Can this be used by everyone?

Yes, even users that don't have javascript (like robots) will be able to use this. It is a great way to present content with specific keywords helping you search result ranking.

Could it be done in another way?

Yes, it is possible to use css to make the text show on hover but then the solution would not work for mobile (where hover don't exist). The positive SEO effect will still be there.

Do we need jQuery for this?

No, this is done without libraries. But in jQuery there is an object called accordeon with methods that does this a little nicer as it animates the collapse of the previous opened elements before it (animated) opens the next.

Can I copy this code to my own site?

Yes, feel free to use (any or all parts of) this code as your own.
